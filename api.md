# API


### 真实访客人数统计

┌──────┬───────────────────────┬──────────────────────────────────┐                                                                                                       
│ 方法 │         路径          │               用途               │                                                                                                       
├──────┼───────────────────────┼──────────────────────────────────┤                                                                                                       
│ POST │ /api/v1/visit         │ 前端页面加载时调用，记录一次访问 │                                                                                                       
├──────┼───────────────────────┼──────────────────────────────────┤                                                                                                       
│ GET  │ /api/v1/visitor/stats │ 获取统计数据                     │                                                                                                       
└──────┴───────────────────────┴──────────────────────────────────┘

```json
{                            
    "code": 200,                                                                                            
    "message": "获取统计成功",          
    "data": {                              
      "today_uv": 42,                         
      "today_pv": 156,                         
      "weekly_uv": 389,                       
      "total_uv": 10234                        
    },                                 
    "error": null                        
}  
```
方案要点
- 识别: 首次访问时在 session 中写入 visitor_id，后续通过 cookie 识别同一访客
- 计数: Redis HyperLogLog（每日 UV + 累计 UV + 每周 UV），每个 key 仅 12KB，百万级不膨胀
- 防刷: session 天然防同一浏览器重复计数；如需额外防刷可以再加 IP 频率限制
